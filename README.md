# 🤖 AI Resume Screening

This project is a simple AI-based Resume Screening system that compares a resume with a job description and calculates a match score.

---

## 🌐 Live Demo
🔗 https://rishu-raj-dev01.github.io/AI-Resume-screening/

---

## 👨‍💻 About Project
This application allows users to upload a resume (PDF) and enter a job description. It then analyzes both and calculates how well the resume matches the job requirements.

---

## 🛠️ Technologies Used
- Python
- Flask
- HTML5
- CSS3
- scikit-learn
- PyPDF2

---

## ⚙️ Features
- Upload resume in PDF format
- Extract text from resume
- Compare with job description
- Generate match percentage score

---

## 📂 How It Works
- User uploads resume
- System extracts text from PDF
- Compares resume with job description
- Displays similarity score using cosine similarity
